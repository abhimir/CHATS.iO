About Us

CHATS.iO is a new social chat site created by @spadoll and @chrismatthieu. The goal of this site is to create a space where conversations can happen dynamically and naturally without logins or registrations. We were aiming to create a cross between Twitter and IRC without setting up accounts or installing software. Chats can be public or private and can be linked to by including any topic/chatroom name in the URL like http://chats.io/party.

You can join multiple chatrooms at once by using the /join command such as: /join party or /join mychat. You can leave a chatroom at any time using the /leave command in the chatroom that you want to leave. You can also change your name at anytime using the /nick command such as: /nick chris.

The technology is written in Node.JS and deployed on Nodester. Node.JS is the latest event-driven technology designed for real-time communications. CHATS.iO uses websockets to push messages to the users' browsers immediately as messages are sent. Chats.io was originally forked from https://github.com/solisoft/Node-Multi-Rooms-Chat
